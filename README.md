# Module 10 Assignment - JavaScript Basic Functions

This project contains the solution for the Module 10 assignment. It includes 8 different JavaScript functions to solve various logical and mathematical problems.

## Functions Included:
1. **calculateDifference**: Returns the subtraction of two numbers.
2. **isOdd**: Checks if a number is odd.
3. **findMin**: Finds the minimum value from an array.
4. **filterEvenNumbers**: Filters out even numbers from an array.
5. **sortArrayDescending**: Sorts an array in descending order.
6. **lowercaseFirstLetter**: Lowercases the first character of a string.
7. **findAverage**: Calculates the average of numbers in an array.
8. **isLeapYear**: Checks if a year is a leap year.
